### 🚀 What I'm building

I run **[CORESYNTH.IO](https://coresynth.io)** — building AI agent infrastructure and full-stack products end to end. A few things I work on:

- **[orcasynth](https://github.com/dragocz1995/orcasynth)** — Self-hosted daemon that orchestrates autonomous AI coding agents (Claude Code, OpenCode, Codex). Autopilot missions, guardrails, REST API, CLI, and a real-time web UI.
- **[asus-ryuo-iv-linux](https://github.com/dragocz1995/asus-ryuo-iv-linux)** — Linux driver for the ASUS ROG RYUO IV AIO cooler LCD, streaming live sensor data over USB HID.
- **Alex** — Agentic AI assistant for auto-parts shops: 26+ tools, TecDoc/VIN decoding, OpenAI-compatible API, and an OpenAI-compatible gateway at `coresynth.io/v1`.
- **Commerce platforms** — Full Next.js 16 + Medusa v2 storefronts and backends for real clients (catalog, orders, Stripe, Packeta shipping, invoicing).
- **Hermes integrations** — Plugging self-hosted agents into the Nous Research Hermes framework over MCP.
