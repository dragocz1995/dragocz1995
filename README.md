I build things end to end out of **[CORESYNTH.IO](https://coresynth.io)** — autonomous AI agents and orchestration daemons, full-stack commerce platforms for real clients, and the occasional low-level Linux driver when the hardware refuses to cooperate.

```text
focus      ·  AI agent infrastructure, agentic tooling, full-stack products
languages  ·  TypeScript, Python, PHP, Go
runtime    ·  self-hosted Linux, daemons, real-time systems
```

**Selected work**

`orcasynth` &nbsp;—&nbsp; self-hosted daemon orchestrating autonomous coding agents (Claude Code, OpenCode, Codex): autopilot missions, guardrails, REST API, CLI, real-time web UI &nbsp;·&nbsp; [→](https://github.com/dragocz1995/orcasynth)
`asus-ryuo-iv-linux` &nbsp;—&nbsp; Linux driver for the ASUS ROG RYUO IV AIO cooler LCD, streaming live sensor data over USB HID &nbsp;·&nbsp; [→](https://github.com/dragocz1995/asus-ryuo-iv-linux)
`Alex` &nbsp;—&nbsp; agentic assistant for auto-parts shops: 26+ tools, TecDoc/VIN decoding, OpenAI-compatible gateway
`commerce` &nbsp;—&nbsp; Next.js 16 + Medusa v2 storefronts & backends — catalog, orders, Stripe, Packeta, invoicing
